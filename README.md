# Dailydoer Todo App

Todo app to manage your tasks.
This is my first react project. My goal was to learn react and tailwind css. 

## Features
- Add todos
- edit todos
- delete todos
- mark todos as completed
- toggle dark and light mode
- todos are saved to localstorage

## Tools used
- react
- vite
- tailwind css
